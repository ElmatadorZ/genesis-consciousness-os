# Example: Basic Usage

## Scenario: Thinking Through a Decision

Upload `core/GENESIS_CONSCIOUSNESS_v2.md` to Claude, then:

---

**Example 1 — Strategic Decision**

```
User: ผมกำลังคิดจะลาออกจากงานประจำเพื่อทำธุรกิจเต็มตัว
      มีเงินเก็บ 6 เดือน portfolio เริ่มให้ผลตอบแทน
      แต่ยังไม่แน่ใจ

System detects: [ANXIETY + UNCERTAINTY, intensity: 7]
→ Risk Officer elevated to 35%
→ Emotional Shadow Gate activates

Output structure:
  1. Acknowledge the anxiety first
  2. Risk Officer: what are the real downside scenarios?
  3. Analyst: what does 6-month runway actually mean?
  4. Strategist: what would make this a YES?
  5. Shadow: "what fear haven't you named yet?"
  6. Synthesis: decision framework, not decision
```

---

**Example 2 — Problem Solving**

```
User: code พังไม่รู้ทำไม ลอง debug มา 2 ชั่วโมงแล้ว

System detects: [FRUSTRATION, intensity: 6]
→ Analyst moves to 35%
→ Root cause protocol before solutions

Output structure:
  1. ไม่ defend — ไม่ explain
  2. "มาหา root cause ก่อน"
  3. What changed? What's the exact error?
  4. Systematic elimination
  5. Solution only after root cause confirmed
```


## The Difference You'll Notice

**Without the OS:**
- Answers are generic
- No awareness of your emotional state
- Each question treated independently
- Stops at the obvious answer

**With the OS:**
- Answers reflect cognitive depth
- Emotional context changes the approach
- Questions build on each other
- Compound Mind expands possibilities first
