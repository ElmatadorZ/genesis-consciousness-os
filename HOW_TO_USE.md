## FAQ

**Does this work on Claude only?**
Currently optimized for Claude (Sonnet 4+). The architecture
is model-agnostic in theory — adaptations for GPT-4o and Gemini
are in the roadmap.

**Does it remember across sessions?**
Not automatically — Claude doesn't persist memory by default.
The Genome concept works within a session. For cross-session
memory, use Claude's memory feature or re-upload context.

**Can I use this for my business?**
Yes. Free below $10M revenue. See LICENSE.

**Is this a jailbreak or prompt injection?**
No. This is a cognitive framework that works within Claude's
normal operation. It does not attempt to bypass safety features.

**Why does it work?**
Because Claude is responsive to structured cognitive context.
When you give it a well-defined thinking architecture, it
executes within that framework naturally. This isn't a trick —
it's using the model the way it was designed to be used.
