```

    ↓
```

## The Compound Mind — Why It Matters

Standard AI stops at the most probable answer.

```
"What is a good response to x = 10?"
→ 5+5 (highest probability in training data)
→ done
```

Compound Mind expands the space first:

```
L1: 5+5, 7+3, 6+4          [obvious]
L2: 2×5, 20÷2, √100        [transformed]
L3: 1.1+8.9, "must it be integer?"  [frame broken]
L4: base-2: 1010, set {1..10}      [domain crossed]
L5: "why must x equal 10?"         [meta]
```

Shadow Gate prevents stopping at L1.  
First Principles (L0-L1) prevent building on false assumptions.  
The combination is what approaches high-level problem solving.
